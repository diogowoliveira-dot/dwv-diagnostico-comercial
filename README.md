# DWV Diagnóstico Comercial

Ferramenta de diagnóstico comercial da DWV — Fórmula de Aceleração para incorporadoras.

## Funcionalidades

- Diagnóstico completo em 6 etapas
- Cálculo de Taxa de Engajamento, Delta de Corretores/Milhão e Corretores Necessários
- 3 rotas estratégicas (Engajamento, Base, Produtividade)
- Organogramas e fluxogramas sem/com Operadora DWV
- Mapeamento de ferramentas com custo mensal
- ROI calculado com economia de ferramentas
- Exportação em PDF e cópia para Pipefy
- Painel consolidado de todas as incorporadoras
- Modo simulação com dados fictícios

## Deploy

Hospedado na Vercel. Push na branch `main` aciona deploy automático.
